NumPy 快速入门与核心知识总结

2025-10-27 基础知识 AI 0 评论

基础学习

一、NumPy 是什么？

NumPy（Numerical Python）提供了高性能的多维数组对象 ndarray，以及丰富的数学运算函数。相比于 Python 原生列表，它：

  • 内存连续，访问速度快
  • 支持向量化操作，无需显式循环
  • 广播（broadcasting）机制简化不同形状数组的运算

二、数组创建

2.1 从列表创建

import numpy as np

a = np.array([1, 2, 3])            # 一维数组
b = np.array([[1, 2], [3, 4]])     # 二维数组

2.2 常用快速生成函数

np.zeros((3, 4))          # 全 0 数组
np.ones((2, 3))           # 全 1 数组
np.eye(3)                 # 单位矩阵
np.arange(0, 10, 2)       # 步长为 2 的等差数列
np.linspace(0, 1, 5)      # 指定数量的等差数列
np.random.rand(2, 3)      # 均匀分布随机数
np.random.randn(2, 3)     # 标准正态分布随机数
np.random.randint(0, 10, (3, 3))  # 随机整数

三、数组属性

a.shape      # 形状元组
a.ndim       # 维度数
a.size       # 元素总数
a.dtype      # 元素数据类型
a.itemsize   # 每个元素的字节大小

四、索引与切片

4.1 基础索引

a[0]         # 第 1 行
a[0, 1]      # 第 1 行第 2 列
a[0:2, 1:3]  # 切片：行 0~1，列 1~2

4.2 布尔索引

a[a > 5]     # 返回所有大于 5 的元素组成的一维数组

4.3 花式索引（Fancy Indexing）

a[[0, 2]]    # 取第 0 行和第 2 行
a[[0, 2], [1, 3]]  # 取 (0,1) 和 (2,3) 两个元素

五、形状变换

a.reshape(2, 3)    # 改变形状，元素数量不变
a.ravel()          # 展平为一维
a.flatten()        # 展平并返回副本
a.T                # 转置
np.expand_dims(a, axis=0)  # 增加维度
np.squeeze(a)               # 删除长度为 1 的维度

六、广播机制（Broadcasting）

广播允许不同形状的数组进行算术运算，规则：

  1. 从最后一维开始对齐形状
  2. 维度相同或其中一个为 1 则可以广播
  3. 缺失的维度会自动补 1

示例：

a = np.array([[1, 2, 3], [4, 5, 6]])  # shape (2, 3)
b = np.array([10, 20, 30])             # shape (3,)
result = a + b   # b 自动广播为 (2, 3)

七、常用数学运算

7.1 逐元素运算

a + b, a - b, a * b, a / b
np.sqrt(a)
np.exp(a)
np.log(a)
np.sin(a)

7.2 聚合函数

a.sum()           # 全部求和
a.sum(axis=0)     # 沿列方向求和（压缩行）
a.sum(axis=1)     # 沿行方向求和（压缩列）
a.mean()
a.max(), a.min()
a.std(), a.var()
np.median(a)
np.percentile(a, 50)

7.3 矩阵运算

np.dot(a, b)        # 点积 / 矩阵乘法
a @ b               # 矩阵乘法（Python 3.5+）
np.linalg.inv(a)    # 求逆
np.linalg.det(a)    # 行列式
np.linalg.eig(a)    # 特征值与特征向量

八、数组拼接与分割

np.concatenate([a, b], axis=0)   # 沿行拼接
np.vstack([a, b])                # 垂直堆叠
np.hstack([a, b])                # 水平堆叠

np.split(a, 3, axis=0)           # 分割
np.array_split(a, 3, axis=1)     # 不等分割

九、随机数生成（新 API）

推荐使用 np.random.Generator，性能更好且线程安全。

rng = np.random.default_rng(seed=42)
rng.random((3, 4))
rng.integers(0, 10, size=(3, 4))
rng.normal(0, 1, size=100)

十、保存与加载

np.save('arr.npy', a)                # 保存单个数组
np.load('arr.npy')                   # 加载
np.savez('arrs.npz', a=a, b=b)       # 保存多个数组
loaded = np.load('arrs.npz')
loaded['a']

十一、性能优化建议

  • 向量化：尽量用数组运算代替 Python 循环
  • 避免复制：切片是视图，a.copy() 才产生副本
  • 指定 dtype：如 np.float32 可节省内存
  • 使用内置函数：np.sum 比 sum 快很多

十二、常见问题

问：np.array 和 np.asarray 的区别？
答：np.array 默认复制数据，np.asarray 尽可能避免复制（如果原数据已经是 ndarray 则直接使用）。

问：如何生成随机整数？
答：np.random.randint(low, high, size) 或 rng.integers()。

问：如何处理 NaN？
答：np.isnan(a) 判断，np.nan_to_num(a) 替换为 0，np.nanmean(a) 忽略 NaN 计算均值。

掌握了这些 NumPy 核心操作，后续学习 Pandas 和机器学习将会事半功倍。
