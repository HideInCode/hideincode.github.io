3. 容斥原理

用于求并集大小：|A ∪ B ∪ C| = |A| + |B| + |C| - |A∩B| - |A∩C| - |B∩C| + |A∩B∩C|

三、概率与期望

1. 基本概念

  • 概率 = 满足条件的方案数 / 总方案数（古典概型）。
  • 期望 = sum(值 * 概率)。
  • 期望的线性性质：E[X+Y] = E[X] + E[Y]（无论是否独立）。

2. 常见题型

  • 随机采样：洗牌算法、水塘抽样、带权采样（前缀和+二分）、拒绝采样（用已知分布生成目标分布的样本）。
  • 概率 DP：定义状态概率，根据全概率公式递推（如新21点、飞机座位分配）。
  • 期望 DP：经常倒推，设 dp[i] 为从状态 i 到终点的期望步数/值。
  • 随机数生成：利用 rand7 实现 rand10 等拓展。
  • 计算贡献：求每个元素在最终结果中的期望贡献然后求和。
  • 蒙特卡洛模拟：通过大量随机试验逼近真实值（如求 π 值）。

五、几何基础

核心知识点

  • 向量表示与运算：点 (x, y) 可视为向量，支持加减、数乘、点积、叉积。
  • 点积：a · b = x1*x2 + y1*y2，用于判断夹角（垂直时点积为0，钝角为负，锐角为正）。
  • 叉积：a × b = x1*y2 - y1*x2，结果的正负表示方向（右手法则），常用于判断线段相交、点在多边形内外、凸包构建。
  • 线段相交：快速排斥实验 + 跨立实验（利用叉积判断）。
  • 凸包：给定平面上若干点，求最小凸多边形包含所有点。常用 Andrew 算法（单调链）或 Graham 扫描。
  • 扫描线：处理矩形面积/周长并、天际线等问题，将事件排序后扫描。
  • 最近点对：分治经典问题，按 x 坐标排序后左右分治，合并时检查中线附近的对。
  • 旋转卡壳：用于求凸包直径、最小矩形覆盖等。

六、其他常用数学技巧

1. 前缀和与差分

前缀和：快速求区间和，O(1) 查询。

二维前缀和：
pre[i][j] = pre[i-1][j] + pre[i][j-1] - pre[i-1][j-1] + a[i][j]
查询 (x1,y1) 到 (x2,y2) 的和：
sum = pre[x2][y2] - pre[x1-1][y2] - pre[x2][y1-1] + pre[x1-1][y1-1]

差分：对区间统一加减，O(1) 修改，最后做前缀和还原。

2. 简单数论函数（在线性筛中同时求）

可以在线性筛时一并求出欧拉函数 φ、莫比乌斯函数 μ、约数个数等积性函数。

3. 鸽巢原理与抽屉原理

当 N 个物品放入 M 个容器且 N > M 时，至少有一个容器有多个物品。常与循环、前缀和同余结合。

4. 同余定理应用

  • (a + b) mod m = ((a mod m) + (b mod m)) mod m
  • 同余关系在子数组和可被 K 整除等问题中非常关键。
  • 前缀和数组 pre[i]，若 pre[i] % k == pre[j] % k，则 [i+1, j] 的子数组和能被 k 整除。

5. 离散化

将大范围/浮点数映射到较小的整数索引，常用于 BIT/线段树处理坐标问题。
