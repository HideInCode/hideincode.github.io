一、数据结构基础

数据结构是算法的基石。刷题时，根据题目给定的数据结构，往往能直接缩小可用的算法范围。

1. 数组与矩阵

  • 核心操作：遍历、双指针、前缀和、差分数组、原地修改
  • 前缀和：快速求任意子数组的和，常用于“子数组和为 K”类问题
  • 差分数组：高效处理区间加法修改，最后前缀和还原
  • 矩阵：常涉及螺旋遍历、旋转图像、搜索二维矩阵，核心是找坐标变换规律
  • 高频题型：螺旋矩阵、杨辉三角、区间合并（LC 56）、缺失数字（LC 268）、消失的数字（LC 448）
  • 必刷题：两数之和（1）、三数之和（15）、盛最多水的容器（11）、接雨水（42）

2. 链表

  • 核心操作：虚拟头节点简化边界、快慢指针、多指针反转
  • 高频题型：反转链表（整体/区间）、环形链表检测与找入口、删除倒数第 N 个节点、合并 K 个有序链表、LRU 缓存
  • 关键技巧：在链表头部加 dummy 节点，避免处理头节点为空的特殊情况
  • 高频题型：反转链表（LC 206）、K个一组翻转（LC 25）、环形链表I/II（LC 141/142）、合并K个升序链表（LC 23）、LRU缓存（LC 146）
  • 必刷题：合并两个有序链表（21）、删除链表倒数第N个节点（19）、回文链表（234）

3. 哈希表 / Map / Set

  • 核心操作：O(1) 查找、去重、计数
  • 典型应用：两数之和、字母异位词分组、最长连续序列、前缀和 + 哈希统计
  • 本质：构建值到索引/频率的映射，用空间换取时间
  • 高频题型：两数之和（LC 1）、字母异位词分组（LC 49）、前K个高频元素（LC 347）
  • 必刷题：最长连续序列（128）、复制带随机指针的链表（138）

4. 栈与队列

  • 栈（LIFO）：括号匹配、表达式求值、辅助栈实现最小栈
  • 单调栈：专门解决“下一个更大元素”类问题，能快速找到左右边界（如接雨水、柱状图中最大的矩形）
  • 队列（FIFO）：BFS 的标配
  • 双端队列：滑动窗口最大值（维护单调递减队列）
  • 高频题型：有效的括号（LC 20）、最小栈（LC 155）、每日温度（LC 739）、滑动窗口最大值（LC 239）
  • 必刷题：逆波兰表达式求值（150）、字符串解码（394）

5. 树与二叉树

  • 遍历：DFS（前/中/后序，递归与迭代）、BFS（层序遍历）
  • 经典问题：最大深度、验证 BST、对称二叉树、最近公共祖先（LCA）、路径总和、构造二叉树
  • 常用技巧：自顶向下传参 vs 自底向上返回；后序遍历在子树间收集信息（如直径、LCA）
  • 高频题型：最大深度（LC 104）、验证二叉搜索树（LC 98）、对称二叉树（LC 101）、最近公共祖先（LC 236）、路径总和（LC 112）
  • 必刷题：翻转二叉树（226）、从前序与中序遍历序列构造二叉树（105）、二叉树展开为链表（114）

6. 堆 / 优先队列

  • 核心功能：动态维护集合中的最值
  • 典型应用：Top K 问题（用大小为 K 的小顶堆）、数据流中位数（双堆法）、合并 K 个有序链表
  • 高频题型：数据流的中位数（LC 295）、合并K个升序链表（LC 23）、前K个高频元素（LC 347）

7. 图

  • 存储方式：邻接表（最常用）、邻接矩阵
  • 基础遍历：DFS（递归/栈）、BFS（队列）
  • 关键算法：
    • 连通块 / 岛屿问题：DFS / BFS / 并查集
    • 最短路径：无权图用 BFS，带正权图用 Dijkstra
    • 拓扑排序：Kahn 算法（BFS）或 DFS 逆后序
    • 动态连通性：并查集（路径压缩 + 按秩合并）
    • 最小生成树：Kruskal（并查集 + 边排序）、Prim
  • 高频题型：岛屿数量（LC 200）、克隆图（LC 133）、课程表（拓扑排序，LC 207）、最短路径（BFS/迪杰斯特拉）、最小生成树、网络流
  • 必刷题：单词接龙（127）、被围绕的区域（130）、冗余连接（684）

8. 高级数据结构

  • 字典树（Trie）：用于字符串前缀匹配、单词搜索
  • 并查集：处理连通性问题，近乎 O(1) 的合并与查询
  • 线段树 / 树状数组：高效处理区间查询与修改（单点/区间）
  • LRU/LFU 缓存：哈希表 + 双向链表

二、算法思想

掌握以下七种算法思想，就能应对 LeetCode 中 90% 的问题。

1. 双指针与滑动窗口

  • 思路：利用数组/链表的有序性或索引差来减少遍历层次
  • 题型细分：
    • 左右指针（对撞指针）：处理有序数组求和/回文判定
    • 快慢指针：链表环检测（Floyd判圈算法）
    • 滑动窗口：处理子串/子数组的连续问题（定长/不定长/单序列/双序列/三指针）
  • 必刷题：两数之和II（167）、验证回文串（125）、删除有序数组中的重复项（26）

2. 二分查找

  • 基础二分：在有序数组中查找目标值或边界
  • 二分答案：当问题可以转化为“是否存在一个 x 满足条件”，且答案是单调的，就对答案值域二分。常见于“最大化最小值”或“最小化最大值”类题目
  • 旋转数组二分：利用局部有序性进行剪枝
  • 必刷题：搜索旋转排序数组（33）、在排序数组中查找元素的第一个和最后一个位置（34）、寻找峰值（162）、爱吃香蕉的珂珂（875）

3. 动态规划（DP）

  • 解题步骤：定义 dp 含义 → 写出转移方程 → 初始化 → 确定遍历顺序 → 举例验证
  • 常见 DP 类型：
    • 线性 DP：爬楼梯、打家劫舍、最长递增子序列
    • 背包 DP：0-1背包、全背包、多重背包、分组背包
    • 区间 DP：最长回文子序列、戳气球
    • 状态机 DP：买卖股票系列（持有/非持有状态转移）
    • 划分DP（分割等和子集）
    • 状压DP / 数位DP / 树形DP（二叉树路径和）
    • 二维 DP：最长公共子序列、编辑距离
    • 计数 DP：求方案总数（通常把 max 换成求和，dp[0] 初始化为 1）
    • 博弈DP / 概率期望DP

4. DFS / BFS 与回溯

  • DFS：深入到底再返回，适合连通块、树的后序遍历、拓扑排序
  • BFS：层层扩散，适合最短路径、最小步数问题
  • 回溯：试探 + 撤销，用于生成所有排列、组合、子集、N 皇后等“列出所有可能”的题目
  • 剪枝：在回溯中提前停止不可能的分支，是提速的关键

5. 贪心算法

  • 适用条件：局部最优能推导出全局最优，往往需要排序预处理
  • 典型场景：区间调度（按结束时间排序）、跳跃游戏（维护最远可达位置）、任务分配
  • 注意：贪心不一定正确，若无法证明，应尝试 DP

6. 分治算法

  • 思想：分解成独立子问题，分别解决后合并
  • 典型：归并排序（同时可解决逆序对问题）、快速排序、平面最近点对

7. 位运算

  • 位运算：异或找唯一数字、n & (n-1) 消去最后一位 1、lowbit 运算
  • 快速幂：O(log n) 计算幂
  • 应用场景：状态压缩、快速幂、找出唯一出现的数字、基础运算（性质/拆位/试填/恒等式/思维）
  • 必刷题：只出现一次的数字（136）、汉明距离（461）、比特位计数（338）

8. 数学

  • 素数筛：埃氏筛 O(n log log n)、线性筛 O(n)
  • 组合数学：卡特兰数、容斥原理、排列组合公式（阶乘 + 逆元）
  • 高频考点：最大公约数/最小公倍数、快速幂、素数筛法（埃氏筛/线性筛）、进制转换、大数运算
  • 扩展考点：数论（模运算、同余）、组合数学（排列组合、卡特兰数）、计算几何、随机算法、概率期望、博弈论

三、题型技巧篇

1. 搜索算法

  • BFS（广度优先搜索）：逐层遍历，利用队列实现；适合找最短路径/最短步数
  • DFS（深度优先搜索）：一路走到底，利用递归/栈实现；适合回溯/拓扑排序/连通性问题
  • 高频题型：网格图（DFS/BFS/综合应用）

2. 字符串算法

  • 核心方法：双指针、哈希表统计
  • 高级算法：KMP（字符串匹配）、Z函数、Manacher（最长回文子串）、字符串哈希、AC自动机、后缀数组、子序列自动机
  • 必刷题：实现 strStr()（28）、最长回文子串（5）、最小覆盖子串（76）

3. 数学与数论

  • 高频考点：最大公约数/最小公倍数、快速幂、素数筛法（埃氏筛/线性筛）、进制转换、大数运算
  • 扩展考点：数论（模运算、同余）、组合数学（排列组合、卡特兰数）、计算几何、随机算法、概率期望、博弈论
  • 必刷题：两数相除（29）、Pow(x, n)（50）、阶乘后的零（172）、快乐数（202）

四、组合题型分类

有些题目会综合多种数据结构和算法，需要灵活运用：

1. 前缀和与差分

  • 思路：前缀和快速计算区间和，差分数组高效处理区间修改
  • 应用场景：和为K的子数组、连续数组的最大和

2. 单调栈

  • 思路：维护栈内元素单调递增或递减
  • 应用场景：基础单调栈、矩形面积问题（贡献法）、最小字典序问题
  • 必刷题：柱状图中最大的矩形（84）、接雨水（42）、每日温度（739）

3. 双堆技巧

  • 思路：使用两个堆（一个大顶堆一个小顶堆）分别维护数据流的两半
  • 应用场景：动态维护中位数、滑动窗口中位数
  • 必刷题：数据流的中位数（295）、滑动窗口中位数（480）

4. 循环排序 (Cyclic Sort)

  • 思路：将每个元素放到它应该在的位置上（通常适用于范围在 0 到 n-1 的数组）
  • 应用场景：寻找缺失/重复的数字
  • 必刷题：缺失数字（268）、找到所有数组中消失的数字（448）

5. 原地链表反转 (In-place Reversal of Linked List)

  • 思路：直接在原链表上操作，改变节点的 next 指针方向
  • 应用场景：反转整个链表、反转部分链表
  • 必刷题：反转链表（206）、反转链表II（92）、K 个一组翻转链表（25）
